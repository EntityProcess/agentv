# Weighted Evaluators

Demonstrates weighted evaluator configurations for prioritizing different evaluation criteria.

## What This Shows

- Assigning weights to evaluators
- Calculating weighted scores
- Prioritizing critical vs optional criteria
- Balanced vs unbalanced weighting schemes

## Running

```bash
# From repository root
bun agentv eval examples/features/weighted-evaluators/evals/dataset.yaml
```

## Key Files

- `evals/dataset.yaml` - Test cases with weighted evaluator configurations
